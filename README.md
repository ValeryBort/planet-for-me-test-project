### Автотесты для проекта Planet for me

#### Как запустить Автотесты

##### Технические требования

- Python 3.6
- [Poetry](https://python-poetry.org/docs/)
- Chrome
- ChromeDriver
- Appium Server
- Android SDK

##### Настройка окружения

- Установить зависимости

        $ poetry update

- Запустить appium-server-gui

- Запустить тест

        $ poetry run pytest
